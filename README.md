# Nostalgia Lógica 🧠

Um jogo de lógica clássico inspirado nas famosas revistas Coquetel, desenvolvido para desafiar o raciocínio dedutivo diretamente no navegador.

## 📝 Sobre o Projeto

O **Nostalgia Lógica** foi um dos meus primeiros projetos autorais. Ele recria a experiência das tabelas de dedução lógica onde o jogador precisa cruzar informações para chegar à resposta correta.

Este repositório é um marco importante na minha jornada como desenvolvedora, representando meus primeiros passos na manipulação do DOM e na construção de regras de negócio com JavaScript puro.

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estruturação das tabelas e conteúdo.
* **CSS3**: Estilização visual inspirada na estética retrô de revistas de passatempo.
* **JavaScript (Vanilla)**: Lógica de marcação, verificação de respostas e interatividade.

## 🚀 Desafios e Evolução (Refatoração)

Como este projeto foi construído no meu primeiro ano de estudos, o código atual é "bruto" e possui muitas repetições (hardcoded), o que torna o carregamento pesado e a manutenção difícil.

Atualmente, estou utilizando este projeto para aplicar novos conhecimentos em **DSM (Desenvolvimento de Software Multiplataforma)** com os seguintes objetivos:

- [ ] **Mecanização de Tabelas:** Criar uma função dinâmica que gere a tabela HTML com base em um objeto JSON, eliminando a necessidade de escrever centenas de linhas de HTML manual.
- [ ] **Automação de Páginas:** Implementar um sistema onde novos desafios possam ser carregados dinamicamente sem a criação de novos arquivos `.html`.
- [ ] **Otimização de Performance:** Limpar redundâncias no JavaScript e melhorar a eficiência dos seletores.

## 🎮 Como Jogar

1. Leia as dicas fornecidas ao lado da tabela.
2. Clique nas células para marcar "X" ou "Confirmado" conforme suas deduções.
3. Use a lógica para preencher toda a grade corretamente!
